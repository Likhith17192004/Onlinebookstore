Online Bookstore

A Spring Boot full-stack web app for book browsing, details, cart management, and checkout.

View Source Code

Core Features

  • Home page with searchable catalog
  • Book details page
  • Add to cart / remove from cart
  • Checkout flow
  • 20 seeded books with cover images
  • Fallback image support for broken cover links

Technology Stack

  • Java 17
  • Spring Boot + Spring MVC + Thymeleaf
  • Spring Data JPA
  • H2 in-memory database
  • Maven Wrapper

How to Run

From repository root:

.\mvnw.cmd spring-boot:run (Windows)

./mvnw spring-boot:run (Mac/Linux)

Then open http://localhost:8080/

Note: GitHub Pages serves this showcase page only. The actual app runs as a Java server locally or on cloud hosting.
